# Project Information

## Private Backend (REST API)
A multiuser platform where users can write a daily journal, and record daily activities.

### Features
- User registration & login
- Write a daily journal
- Activity Tracker
- Mood Tracker
- Task Manager

- Database management with MongoDB (NoSQL)
